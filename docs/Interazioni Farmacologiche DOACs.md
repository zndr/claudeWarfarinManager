## Tabelle delle Principali Interazioni Farmacologiche dei DOAC

Di seguito sono riportate le tabelle riassuntive delle principali interazioni tra ciascun DOAC e altri farmaci, con evidenziazione nella colonna specifica delle interazioni pericolose che richiedono controindicazione o particolare attenzione clinica.

***

### **DABIGATRAN (Pradaxa)**

Dabigatran è substrato esclusivo della **P-glicoproteina (P-gp)** e **non è metabolizzato dal sistema del citocromo P450**. L'eliminazione renale è dell'80%.[^1]

| Ciclosporina | ↑ esposizione significativa | **CONTROINDICATO** | ⚠️ **SÌ** |
| Itraconazolo | ↑ esposizione (simile ketoconazolo) | **CONTROINDICATO** | ⚠️ **SÌ** |
| Dronedarone | ↑ AUC/Cmax ~2.4/2.3x | **CONTROINDICATO** | ⚠️ **SÌ** |
| Glecaprevir/pibrentasvir | ↑ esposizione, ↑ rischio sanguinamento | **CONTROINDICATO** | ⚠️ **SÌ** |
| **INIBITORI MODERATI P-gp (RIDUZIONE DOSE)** |  |  |  |

Apixaban è substrato della **P-glicoproteina (P-gp)** e del **CYP3A4** (<32% metabolismo). Eliminazione renale 27%.[^4]


| **Classe farmacologica / Farmaco** | **Effetto su apixaban** | **Raccomandazione clinica** | **Interazione pericolosa** |
| :-- | :-- | :-- | :-- |
| **INIBITORI FORTI P-gp + CYP3A4 (NON RACCOMANDATO)** |  |  |  |

## Note Generali sulle Interazioni Pericolose

### Criteri per definire un'interazione "pericolosa"

Un'interazione è considerata **pericolosa** quando:

1. **Aumenta significativamente il rischio emorragico** (aumento >50% dell'esposizione al DOAC o interazione farmacodinamica additiva con rischio documentato di sanguinamento maggiore)
2. **Riduce significativamente l'efficacia anticoagulante** (riduzione >30% dell'esposizione, con rischio di eventi tromboembolici)
3. **Richiede controindicazione assoluta** nei RCP ufficiali EMA
4. **Richiede riduzione obbligatoria della dose** per evitare tossicità
5. **Ha evidenza clinica di eventi avversi gravi** da studi osservazionali o case report[^7][^2][^8]

### Interazioni comuni pericolose per tutti i DOAC[^9][^10][^11]

- **Anticoagulanti multipli** (combinazione DOAC + AVK o DOAC + EBPM)
- **Doppia/tripla terapia antipiastrinica** senza indicazione appropriata
- **FANS ad alte dosi croniche** (specialmente in pazienti anziani o con IR)
- **Induttori enzimatici forti** (rifampicina, antiepilettici classici) → rischio trombotico
- **Inibitori forti combinati P-gp + CYP3A4** (antimicotici azolici, inibitori proteasi HIV) → rischio emorragico


### Fattori che amplificano le interazioni pericolose[^2][^8][^10]

- **Insufficienza renale moderata-severa** (specialmente con dabigatran ed edoxaban)
- **Età ≥75 anni**
- **Peso corporeo <60 kg**
- **Politerapia** (≥5 farmaci)
- **Malattia epatica Child-Pugh B-C**
